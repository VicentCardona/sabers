📚 Els Sabers Bàsics (Què hem d'aprendre?)

Segons la normativa vigent, el curs s'estructura en tres grans blocs de coneixement. Aquí trobaràs la teoria, les simulacions i quin projecte pràctic farem per assolir cadascun d'ells.

Saber Bàsic A

Algorísmia i Programació

Aquest bloc se centra en la resolució de problemes (divideix i guanyaràs), l'ús de bucles imbricats, les funcions, la reutilització de codi i el paradigma de la Programació Orientada a Objectes (POO).

Suport Teòric

  • 📗 Teoria: Estructures de Resolució de Problemes
  • 📗 Conceptes de POO: Classes, Herència i Polimorfisme
  • ▶️ Simulació: Diagrames de classes interactius

🎯 Com ho posem en pràctica?

1. El Repte Mario (CS50): Utilitzarem C per comprendre l'algorísmia pura basant-nos en el curs de Harvard.
2. Llibre Python Crash Course: Crearem els nostres propis objectes i mòduls en Python per consolidar la POO.

Saber Bàsic B

Tractament de Dades

Comprèn l'estudi del Model Entitat-Relació, les regles d'integritat per a Bases de Dades SQL (Formes Normals), els fitxers externs (JSON, CSV) i l'accés a dades mitjançant APIs i connexions web.

Suport Teòric

  • 📗 Teoria: El model Relacional i les Comandes SQL (DDL/DML)
  • 📗 Entenent els fitxers d'emmagatzematge: CSV i JSON
  • ▶️ Interactiu: Executa consultes SELECT sobre la nostra BBDD web

🎯 Com ho posem en pràctica?

3. Tractament Massiu: Connectarem aplicacions Python a les APIs i descarregarem datasets governamentals.
4. Back-End Actiu: Farem servir Django per crear bases de dades reals integrades a una pàgina web on els usuaris injectaran informació.

Saber Bàsic C

Aplicació en Entorns Simulats (i Publicació)

Aplicació pràctica dels coneixements a branques com videojocs, IA, domòtica o tractament massiu. Inclou la necessitat fonamental de documentar de forma clara per a l'usuari final (Saber Bàsic D de Primer, actiu a Segon).

Suport Teòric i Exploració

  • 📗 Guia per comentar i modular el codi de forma neta
  • 🌐 Branques tecnològiques: El Futur de l'IoT (Internet of Things)

🎯 Com ho posem en pràctica?

5. El Projecte Final Lliure: L'alumne escollirà un entorn simulat (videojoc, anàlisi de dades massiva o domòtica) per implementar una solució global que reuneixi tots els Sabers, i serà responsable de redactar i publicar el manual d'usuari (UX).
